# 作者简介

**杨艺韬**，资深工程师，长期负责大规模业务系统的架构设计与工程落地。工作中深耕 Playwright 自动化、Electron 桌面应用、浏览器内核定制与 JavaScript 注入等前沿技术方向，在复杂架构、自动化工程与系统级底层技术领域积累了丰富的实战经验。

出于对 AI Agent 系统架构的持续热情，杨艺韬在业余时间逐行审读 OpenClaw 全部源码，历经 69 轮迭代审校，将一个生产级运行时系统的每一个设计决策还原为可追溯、可验证的工程叙事，最终写成本书。

他相信：理解"为什么"比掌握"怎么做"更持久、更有价值。好的技术书不应该只教读者使用工具，而应该帮助读者建立设计直觉——那种面对未知问题时，能从第一性原理出发做出正确判断的能力。
